How to Generate Clinical Report?
Enable selection of variant(s) of interest to be reported into a clinical report with the uniquely built report template from each Hub.
Step 1: The User can tick the checkbox for the variant within the result table, click on the Save for Reporting button before being prompted with a box to enter the desired notes
Step 2: Upon clicking the Save button, the User will then be able to see the pinned and the custom note within the "Variants Selected for Reporting" section right below the Clinical Annotations section. The user will then be able to click on the Generate Report button to automatically download the clinical report with the selected variant, OR delete the selected variant using the delete icon at the top right corner.
