How to Use The Advanced Filter?
Enable query of a specific string of letters/words per available columns within the result table, and can be combined for sequential filtering purposes. This feature uses the "AND" logic, so when the parameters are added, only the specific entry(ies) with the parameters will be shown. To note, when the parameters were added sequentially, where one parameter is filtered first, then the second parameter will only filter the currently filtered/shown result in the UI. Hence, if combined parameters are to be used, the User is encouraged to enter the parameters first before clicking on the "Filter" button. The steps will be as follows:
Step 1: Users can click on the collapsible Advanced Filter Term card.
Before Collapse
After Collapse
Step 2: Users can choose a parameter from the available columns within the result table.
Step 3: Upon choosing the parameters, Users can then click on the "Add Filter" button, to which the query section for the specific parameter will appear under the search box.
Before
After
Step 4: By clicking on the entry field, Users will be able to see the values present within the chosen column from the results. Note that only values present within the result table will be shown/queryable.
Step 5: Upon choosing the parameter's value, Users can choose another filtering parameter by redoing step 2 to step 4 above.
Step 6: Users can then click on the "Filter" button below the parameters to query the specific entry(ies) possessing the values entered. If desired, Users can also replace the parameters by clicking on the delete button right next to the parameter field, and redo the step 2 to step 6 steps.
