How to Add Custom Annotation?
Enable pinning and adding custom notes/annotations from Users for particular variant(s) of interest for further analysis/discussion. The steps to utilise this function are described below:
Step 1: The user can tick the checkbox for the variant within the result table and click on the Annotate button. The user will then be prompted with a box to enter the desired notes.
Step 2: Upon clicking the Save button, the annotated variant(s) will be highlighted in the result table.
Step 3.1: The annotated variant(s) will also be visible under the "Clinical Annotations" section, right below the result table.
Step 3.2: In this "Clinical Annotations" section, there are three actionable buttons:
View Detail Variants
Add to reporting
Delete Annotations
View detail variants: Clicking this button will filter the result table to only show the chosen annotated variant(s).
Add to reporting: This button behaves the same as the "Save for Reporting" button, which will be discussed in the next section. However, this button is a shortcut to enable Users to add the variants directly for clinical report generation.
Delete Annotations: Clicking this button will delete the annotated variants and their annotations from this "Clinical Annotations" section. This feature will also remove the highlighted borders surrounding the previously annotated variants in the result table.
