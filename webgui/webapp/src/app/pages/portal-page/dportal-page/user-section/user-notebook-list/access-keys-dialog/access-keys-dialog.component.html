Access Keys

You can use the following keys to access S3 to upload/download data.
Do not share following keys.
export AWS_ACCESS_KEY_ID="{{ accessKeyId }}"
export AWS_SECRET_ACCESS_KEY="{{ secretAccessKey }}"
export AWS_SESSION_TOKEN="{{ sessionToken }}"
S3 URL is
To access S3 using AWS CLI, run the following command: 
        aws s3 ls {{ s3uri }}
To copy files to s3, run the following command: 
        aws s3 cp file.txt {{ s3uri }}
To copy files from S3, run the following command: 
        aws s3 cp {{ s3uri }}/file.txt .
