GaspiFS Commands Reference

GaspiFS is a secure CLI for managing your projects. You can view projects, explore files within projects, and move files from project storage into your notebook. Below is a list of available commands to help you work with your data efficiently.

Please note that you will have to login first to use the commands. If you need to download files from a notebook, without interactive CLI access, please use login command (details in Authentication tab) to authenticate your session first. To do this, you can open a new terminal window and the logged in session will be active in the notebook. Username and password are the same GASPI credentials.

To use CLI commands within the Notebook cells use the following syntax. Note the exclamation mark at the beginning. 

!gaspifs projects

List Projects

Display all genomic projects you have access to with their names and descriptions. 

gaspifs projects
Project 1:
    Name: "Whole Genome Analysis"
    Description: "WGS data for rare disease study"
Project 2:
    Name: "Exome Sequencing"
    Description: "Clinical exome analysis pipeline"

User Space Files

List files in your personal workspace (saved queries, analysis results). 

gaspifs files
File    1: "saved-queries/variant-filter.json"
File    2: "saved-queries/population-analysis.json"

Project Files

List genomic data files within a specific project. -p specifies the project name. 

gaspifs files -p "Whole Genome Analysis"
File    1: "samples.vcf.gz"
File    2: "samples.vcf.gz.tbi"
File    3: "metadata.json"
File    4: "quality-control.txt"

Download from Project

Download files from a specific genomic project to your workspace.. -p specifies the project, -f specifies the file, and -d specifies the destination directory. 

gaspifs download -f samples.vcf.gz -d files -p "Whole Genome Analysis"

Download from User Space

Download files saved in your personal workspace. -f specifies the file, -d specifies the destination directory. 

gaspifs download -f "saved-queries/variant-filter.json" -d files

Login

Authenticate with GaspiFS to start using commands and access your files. 

gaspifs login

Logout

Sign out of GaspiFS to end your session and secure your access.

gaspifs logout

Get Help

Shows available commands within the terminal.

gaspifs help
Use quotes around project names and filenames that contain spaces. File extensions like .vcf.gz, .json, .txt help identify data types.
